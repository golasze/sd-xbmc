# Jak zainstalować plugin.video.polishtv.live #

autorem artykułu jest **broman**, a poniższa instrukcja została zaporzyczona z jego blogu http://broman.pl/blog

# Instalacja krok po kroku: #

Pobieramy plik zip z repozytorium http://sd-xbmc.googlecode.com/files/repository.sd-xbmc.org-1.0.2.zip

Instalujemy pobrane repozytorium:

Wybieramy _**System > Settings**_

![http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-000.jpg](http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-000.jpg)

Dalej _**Add-ons**_

![http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-001.jpg](http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-001.jpg)

Dalej _**Install from zip file**_

![http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-002.jpg](http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-002.jpg)

Dalej wybieramy wcześniej ściągnięty plik **_repository.sd-addons.eu.zip_**

![http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-003.jpg](http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-003.jpg)

Widzimy że nasze repozytorium zostało dodane (prawy dolny róg)

![http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-004.jpg](http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-004.jpg)

Wybieramy _**Get Add-ons**_

![http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-005.jpg](http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-005.jpg)

Dalej wybieramy nasze repozytorum **_SD XBMC Add-ons_**

![http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-006.jpg](http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-006.jpg)

Dalej _**Video Add-ons**_

![http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-007.jpg](http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-007.jpg)

I nasz poszukiwany plugin **_Telewizja Polska ..._**

![http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-008.jpg](http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-008.jpg)

Wybieramy _**Install**_

![http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-009.jpg](http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-009.jpg)

Czekamy aż plugin się ściągnie

![http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-010.jpg](http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-010.jpg)

I widzimy że jest już zainstalowany

![http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-011.jpg](http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-011.jpg)

Teraz z ekranu startowego juz możemy wybrać _**Videos > Add-ons**_

![http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-012.jpg](http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-012.jpg)

I widzimy nasz plugin

![http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-013.jpg](http://broman.pl/webdocs/image/xbmc-tvp/xbmc-instalacja-tvp-013.jpg)
